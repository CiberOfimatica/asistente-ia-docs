Política de Privacidad

Esta aplicación utiliza la API de OpenAI exclusivamente para procesar texto bajo petición del usuario. No se almacena, comparte ni analiza información personal. Los datos se usan solo durante la sesión activa del usuario y no se guardan en servidores externos.
